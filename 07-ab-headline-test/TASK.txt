        Ticket 07: Run a Simple Headline A/B Test

        Scenario
        Marketing wants to compare two headlines. A visitor must keep the same variant during repeat visits, and every impression and CTA click must include that variant.

        Concepts
        arrays, random selection, localStorage, DOM text updates, experiment metadata

        Instructions
        1. Open index.html in a browser.
        2. Open script.js in Visual Studio Code or Visual Studio.
        3. Complete each TODO without copying the answer first.
        4. Refresh index.html after every change.
        5. Use the browser console to inspect errors and window.dataLayer.

        Challenge
        Add a third variant C and change the code so all variants still have an equal chance of assignment.

        Definition of done
        - 
A or B is assigned- Reload keeps the same assignment- Impression and click events include the variant- Reset creates a new assignment opportunity
