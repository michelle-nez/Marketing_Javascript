        Ticket 02: Validate and Track a Lead Form

        Scenario
        A sales form should stop incomplete submissions, show a helpful message, and record a lead event without sending personal information into analytics.

        Concepts
        submit events, preventDefault, trim, validation, privacy-safe event data

        Instructions
        1. Open index.html in a browser.
        2. Open script.js in Visual Studio Code or Visual Studio.
        3. Complete each TODO without copying the answer first.
        4. Refresh index.html after every change.
        5. Use the browser console to inspect errors and window.dataLayer.

        Challenge
        Add a checkbox for newsletter permission, then include only the true/false permission value in the analytics event.

        Definition of done
        - 
Empty submissions are blocked- A complete form shows success- The event contains interest but no name or email- The form resets after success
