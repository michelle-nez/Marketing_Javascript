        Ticket 10: Add Consent-Aware Analytics

        Scenario
        A site must not record marketing events until the visitor grants analytics permission. Rejected visitors should still be able to use the page.

        Concepts
        state management, localStorage, queues, conditional logic, event tables

        Instructions

        2. Open script.js in Visual Studio Code or Visual Studio.
        3. Complete each TODO without copying the answer first.
        4. Refresh index.html after every change.
        5. Use the browser console to inspect errors and window.dataLayer.
